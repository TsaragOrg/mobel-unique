# Change Requests

Accepted specs are frozen by default.

To change an accepted spec, create a change request here before editing the accepted spec.

Recommended filename:

```text
CR-SPEC-0001-short-title.md
```

Each change request should include:

- Target spec id
- Reason for change
- Proposed change
- Impact on plans, tests, roadmaps, API, database, worker, or UI
- Approval note
